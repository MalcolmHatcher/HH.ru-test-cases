# Тест-кейсы для сайта hh.ru

## 1. Регистрация пользователя (Registration) 

| ID     | Описание                          | Шаги                                                                 | Ожидаемый результат                          | Тип        |
|--------|-----------------------------------|----------------------------------------------------------------------|----------------------------------------------|------------|
| REG-01 | Успешная регистрация соискателя  | 1. Перейти на главную.<br>2. Нажать "Зарегистрироваться".<br>3. Ввести валидный номер.<br>4. Подтвердить код. | Пользователь зарегистрирован, перенаправлен в кабинет. | Позитивный |
| REG-02 | Регистрация с невалидным номером | 1. Ввести некорректный номер.<br>2. Продолжить.                     | Ошибка "Неверный формат".                   | Негативный |
| REG-03 | Регистрация без принятия соглашения | 1. Ввести валидные данные.<br>2. Не отметить чекбокс соглашения.<br>3. Нажать "Зарегистрироваться" | Предупреждение о необходимости принять соглашение. | Негативный |

## 2. Вход в систему (Login) 

| ID     | Описание                          | Шаги                                                                 | Ожидаемый результат                          | Тип        |
|--------|-----------------------------------|----------------------------------------------------------------------|----------------------------------------------|------------|
| Login-01 | Успешный вход с валидными credentials | 1. Перейти на hh.ru.<br>2. Нажать "Войти".<br>3. Ввести зарегистрированный номер телефона и код из SMS. | Пользователь авторизован и перенаправлен в личный кабинет. | Позитивный |
| Login-02 | Вход с неверным кодом | 1. Ввести номер.<br>2. Ввести неправильный код из SMS | Отображается ошибка "Неверный код". Доступ запрещен. | Негативный 
| Login-03 | Восстановление доступа при забытом пароле/коде | 1. Нажать "Забыли пароль?".<br> 2. Ввести номер и подтвердить | Отправлен новый пароль, пользователь может авторизоваться. | Позитивный
